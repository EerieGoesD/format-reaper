Install FFmpeg
Windows
macOS
Linux
Quickest: one command in PowerShell
winget install -e --id Gyan.FFmpeg
  1. Press Win + X, choose Terminal (or PowerShell).
  2. Paste the command above and press Enter. Accept the terms when prompted.
  3. Close this dialog and click the Re-check / refresh button.
Or download the zip manually
  1. Open gyan.dev/ffmpeg/builds.
  2. Under release builds, download ffmpeg-release-essentials.zip.
  3. Extract it. Inside there's a bin folder containing ffmpeg.exe and ffprobe.exe.
  4. Move the extracted folder somewhere permanent, e.g. C:\ffmpeg.
  5. Press Win, type Environment Variables, open it, edit Path, add C:\ffmpeg\bin.
  6. Close every PowerShell window, then click Re-check.
With Homebrew
brew install ffmpeg
  1. Open Terminal.
  2. If you do not have Homebrew, install it first from brew.sh.
  3. Run the command above.
  4. Click Re-check in Format Reaper.
Debian, Ubuntu, Pop!_OS
sudo apt install ffmpeg

sudo dnf install ffmpeg

sudo pacman -S ffmpeg
